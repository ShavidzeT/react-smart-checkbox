# react-smart-checkbox
**Installation:**
*  npm
```bash
npm i -P react-smart-checkbox
```
*  yarn
```bash
yarn add react-smart-checkbox
```
**How to use:**
*  import component
```js
import ReactSmartCheckbox from 'react-smart-checkbox'
```
* use component
```js
<ReactSmartCheckbox
  name='fruits'
  options={['orange', 'apple', 'banana']}
/>
```
